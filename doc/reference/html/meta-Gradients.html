enum MetaGradientType

typedef enum {
  META_GRADIENT_VERTICAL,
  META_GRADIENT_HORIZONTAL,
  META_GRADIENT_DIAGONAL,
  META_GRADIENT_LAST
} MetaGradientType;

META_GRADIENT_VERTICAL

 Vertical gradient

META_GRADIENT_HORIZONTAL

 Horizontal gradient

META_GRADIENT_DIAGONAL

 Diagonal gradient

META_GRADIENT_LAST

 Marks the end of the MetaGradientType enumeration
